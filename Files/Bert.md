# 演变史
## one-hot存在的问题
```
维度灾难：容易受维数灾难的困扰，每个词语的维度就是语料库字典的长度；
离散(稀疏)问题：因为 one-Hot 中，句子向量，如果词出现则为1，没出现则为0，但是由于维度远大于句子长度，所以句子中的1远小于0的个数；
维度鸿沟问题：词语的编码往往是随机的，导致不能很好地刻画词与词之间的相似性。
```
## wordvec存在的问题
```
多义词问题:因为 word2vec 为静态方式，即训练好后，每个词表达固定；
SKIP AND CBOW
```
